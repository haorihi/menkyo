## 2026-05-01 - [Add ARIA labels and focus styles to icon-only buttons]
**Learning:** Icon-only buttons lacking `aria-label` and visual focus states are a common pattern in the app that hampers keyboard and screen reader accessibility. Adding Tailwind `focus-visible` classes significantly improves keyboard navigation visibility without compromising the visual design for mouse users. Playwright's `locator` `:has-text()` is a powerful feature, but remember that standard DOM queries like `document.querySelector` do not support these custom pseudo-classes inside `page.evaluate()`.
**Action:** Always check icon-only interactive elements for `aria-label` and `focus-visible` classes. Use standard JavaScript string matching (e.g. `Array.from(document.querySelectorAll('button')).find(b => b.textContent.includes('...'))`) inside `evaluate()` blocks instead of Playwright-specific selectors.
